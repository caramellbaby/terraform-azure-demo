# Terraform Azure Infrastructure Automation

Dette prosjektet ble utviklet som en del av avsluttende prosjekt ved TISIP Fagskole innen IT-drift.

Prosjektet viser hvordan Azure-infrastruktur kan automatiseres ved hjelp av Terraform og GitHub Actions.

Teknologier:
- Terraform
- Microsoft Azure
- GitHub Actions
- Microsoft Entra ID
- OIDC-autentisering

Prosjektet demonstrerer:
- Infrastructure as Code (IaC)
- Automatisk utrulling av Azure-ressurser
- CI/CD med GitHub Actions
- Sikker autentisering uten passord ved hjelp av OIDC

Prosjektet ble utviklet for lærings- og demonstrasjonsformål i et testmiljø.

Prosjektet ble utviklet i et Azure-testmiljø og er ment som en demonstrasjon av Infrastructure as Code (IaC) og automatisering av skyinfrastruktur.
